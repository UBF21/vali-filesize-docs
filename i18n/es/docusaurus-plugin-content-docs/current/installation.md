# Instalación

## .NET CLI

```bash
dotnet add package Vali-FileSize
```

## Package Manager

```bash
Install-Package Vali-FileSize
```

## PackageReference

```xml
<PackageReference Include="Vali-FileSize" Version="1.1.0" />
```

## Requisitos

| Target | Versión mínima |
|---|---|
| .NET Standard | 2.0 o 2.1 |
| .NET | 6.0, 7.0, 8.0 o 9.0 |

## Espacios de nombres

Agrega las siguientes directivas `using` donde uses la librería:

```csharp
using ValiFileSize;
using ValiFileSize.Core.Enums;
using ValiFileSize.Core.Extensions; // para métodos de extensión
```
