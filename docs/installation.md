# Installation

## .NET CLI

```bash
dotnet add package Vali-FileSize
```

## Package Manager

```bash
Install-Package Vali-FileSize
```

## PackageReference

```xml
<PackageReference Include="Vali-FileSize" Version="1.1.0" />
```

## Requirements

| Target | Minimum version |
|---|---|
| .NET Standard | 2.0 or 2.1 |
| .NET | 6.0, 7.0, 8.0, or 9.0 |

## Namespace

Add the following using directive wherever you use the library:

```csharp
using ValiFileSize;
using ValiFileSize.Core.Enums;
using ValiFileSize.Core.Extensions; // for extension methods
```
